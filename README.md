# Earth Vanguard

個人製作
C言語およびC++の学習を始めて1年目に制作した、オリジナルの2Dシューティングゲームです。

## 🎮 ゲーム概要
- **ジャンル:** 2Dシューティング
- **開発環境:** C++ / GSLIB

## ✨ アピールポイント（こだわった点）
* **4種類の自機による多様なプレイ体験**
  
  性能や特性が異なる4種類の機体を実装しました。プレイヤーが自分のプレイスタイルに合わせて機体を選択し、何度でも新鮮な感覚で遊べるように工夫しています。
  <img width="1232" height="551" alt="スクリーンショット 2026-04-02 160138" src="https://github.com/user-attachments/assets/98c14109-61bf-4948-871d-7ce571e9c6a7" />

* **相性を意識した7種類の敵キャラクターと戦略性**
  
  特徴の異なる7種類の敵を配置し、選択した自機によって「得意な敵・不得意な敵」が生まれるように設計しました。ただ撃つだけでなく、機体の特性を活かした戦略的なプレイが求められるゲームバランスを目指しました。
